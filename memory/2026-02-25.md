**Error Reporting Clarity (User Instruction):**
When an error message appears (tool errors, failed commands, broken setups, etc.), always be explicit about:
1. **Was it fixed?** → Yes/No, and if yes, how.
2. **Is it permanent?** → If permanent (e.g., data loss, unrecoverable state), escalate immediately and offer restoration steps.
