## Key pitfalls/bugs found
### 1) `gog auth credentials` command is wrong (path + subcommand)
In v0.11.0, the command is:
- `gog auth credentials set <credentials.json>`

But both **MEMORY.md** and the **cron payload** use:
- `gog auth credentials /data/workspace/gog_client_secret.json`  ← likely fails

Also: should add `--no-input` to prevent hanging in isolated runs.

### 2) Google Tasks list call won’t include completed tasks + only returns first page
Cron payload uses:
- `gog tasks list <tasklistId> ... --json --no-input`

Default behavior:
- **does NOT include completed tasks** unless `--show-completed`
- **defaults to max=20** unless `--all`

So the sheet will often **never get “done ✅” updates**, and may miss tasks beyond the first 20.

### 3) Minor robustness notes (not blockers)
- Sheet range fixed at `Tasks!A2:K1000` → could miss rows if you grow beyond 1000.
- “Stop at first blank ID in column A” can prematurely stop if there are accidental blank lines.
